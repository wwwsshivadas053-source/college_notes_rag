How It Works

Ingestion

PyMuPDF extracts selectable text from uploaded PDFs. Text is normalized and split into overlapping chunks for retrieval.

Retrieval

The app builds a per-user vector index. It uses Gemini embeddings with FAISS when available and falls back to local TF-IDF retrieval.

Answering

Gemini receives only the retrieved context and the question. If no API key is configured, the app returns the strongest matching excerpt.
